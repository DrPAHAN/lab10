# Лабораторная работа №10: DNS и Traceroute
---

### Задачи:
1. Выполнить DNS-запросы для списка доменов
2. Получить IP-адреса доменов
3. Выполнить traceroute для каждого IP-адреса
4. Сохранить результаты в CSV-файл

---

### Требования:
- macOS 12 (Monterey) или другая Unix-подобная ОС
- Python 3.6+
- Утилиты `dig` и `traceroute` (встроены в macOS)

# Ручной запуск
## 1. DNS-запрос
```
dig +short google.com
# Вывод: 142.250.21.113
```
## 2. Traceroute
```
traceroute -m 10 -w 2 142.250.21.113
```
-m 10 — максимум 10 хопов (прыжков)
-w 2 — таймаут 2 секунды на каждый хоп

## 3. CSV-экспорт
Результаты сохраняются в формате CSV с колонками:
Домен — имя домена
IP-адрес — полученный IP
Результат traceroute — полный вывод трассировки
